# 开个花店

《开个花店》是一款移动端优先的轻松种花与花束创作小游戏。

玩家在室外花圃中种植和照料花朵，收获花材后，通过辅助式 DIY 制作花束并交付顾客订单，获得代币后继续购买种子、扩展花材与创作空间。

## 当前目标

在一周、约 10 至 16 小时投入内，完成一个可公开试玩的垂直切片：

```text
接取开业第一单
→ 种植 4 种花材
→ 浇水、施肥、除虫并收割
→ 制作首束花
→ 交付订单并获得代币
→ 使用代币开始下一轮种植
```

第一阶段重点验证：

1. 种花过程是否具有期待感与收获爽感。
2. 辅助式 DIY 是否既有参与感，又能稳定产出漂亮花束。
3. 交付订单后的奖励是否足以驱动玩家再次种植。

## 首束花

新手首束花暂定为清新奶油色花束，视觉目标参考本地文件：

`风格素材/开个花店素材/Gemini_Generated_Image_kct3ivkct3ivkct3.png`

首束花使用四种可种植花材：

- 粉色大丽花：主花
- 香槟色洋牡丹：配花
- 蓝色飞燕草：线条花
- 白色小雏菊：填充花

绿叶、包装纸和丝带由首个顾客订单提供。

## 平台与技术方向

- 主工程：Cocos Creator 3.8.8 + TypeScript
- 首个公开试玩：移动端优先 Web 构建
- 后续发布目标：微信小游戏、抖音小游戏
- 首版存档：平台本地存档；暂不接入后端、账号、支付或真实广告
- 旧 Phaser 原型：保留在 `prototypes/phaser-bouquet/`，仅作交互决策记录

选择 Cocos 是为了让同一套核心工程可以面向 Web、微信和抖音小游戏构建，同时保留适合 2D 游戏的场景、动画、资源和触屏工作流。Web 仍作为最低成本、最快迭代的首个公开试玩渠道。

## 当前状态

已完成首个 Phaser 可玩技术原型，并将项目正式迁移到 Cocos Creator。当前 Cocos 线框版本已经可以完整体验“开业第一单”：

- 接取订单并领取四种系统赠送种子。
- 在 6 块新手土地完成种植、批量浇水、施肥、除虫、免费成熟和收割。
- 将收获花材拖入花束，按花艺角色自动吸附并完成整理。
- 交付花束获得金币，并进入奖励结算与下一轮种植。
- 第一单、库存、订单奖励和花束规则均可脱离 Cocos 独立测试。

首单完成后还可以继续体验第二轮引导：

- 集中领取成长金币与首次免费发现机会。
- 必定发现新花种“珊瑚玫瑰”，并使用新旧花种完成第二轮种植。
- 从“暖阳心愿”和“春日来信”中选择本轮花束目标。
- 解锁一键收割，获得足以制作两种候选花束的库存。
- 快捷制作并交付所选花束，升级至 Lv.3 并解锁两块新土地。

当前程序绘制的花朵与界面只用于流程和交互验证，明确不代表最终 UI 或美术品质。下一阶段将重点验证奖励连击、发现新花、一键收割和花束选择是否真正产生爽感，并逐步接入正式花圃、花材与花束资产；本地存档仍未实现。
