# 开个花店

《开个花店》是一款移动端优先的轻松种花与花束创作小游戏。

玩家在室外花圃中种植和照料花朵，收获花材后，通过辅助式 DIY 制作花束并交付顾客订单，获得代币后继续购买种子、扩展花材与创作空间。

## 当前目标

在一周、约 10 至 16 小时投入内，完成一个可公开试玩的垂直切片：

```text
接取开业第一单
→ 种植 4 种花材
→ 浇水、施肥、除虫并收割
→ 制作首束花
→ 交付订单并获得代币
→ 使用代币开始下一轮种植
```

第一阶段重点验证：

1. 种花过程是否具有期待感与收获爽感。
2. 辅助式 DIY 是否既有参与感，又能稳定产出漂亮花束。
3. 交付订单后的奖励是否足以驱动玩家再次种植。

## 首束花

新手首束花暂定为清新奶油色花束，视觉目标参考本地文件：

`风格素材/开个花店素材/Gemini_Generated_Image_kct3ivkct3ivkct3.png`

首束花使用四种可种植花材：

- 粉色大丽花：主花
- 香槟色洋牡丹：配花
- 蓝色飞燕草：线条花
- 白色小雏菊：填充花

绿叶、包装纸和丝带由首个顾客订单提供。

## 平台与技术方向

- 首发平台：移动端优先 Web 游戏
- 部署目标：静态站点公开部署
- 计划技术栈：TypeScript + Phaser + Vite
- 首版存档：浏览器本地存档
- 后续方向：核心循环验证后，再评估微信或抖音小游戏迁移

选择 Web 首发，是为了以最低开发、部署和审核成本跑通完整流程。首版不接入后端、账号、真实广告或支付。
